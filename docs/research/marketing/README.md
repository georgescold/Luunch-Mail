## Deux catégories à ne pas confondre

- **Plateformes de marketing automation** (Klaviyo, ActiveCampaign) : pour les **marketeurs**, no-code, campagnes + flows + segmentation + CRM/CDP, facturées **au nombre de contacts**.
- **Infrastructure / API d'e-mail** (Resend, type SendGrid/Postmark/Mailgun) : pour les **développeurs**, envoi **par code**, cœur **transactionnel** + newsletters via Broadcasts, facturée **au volume d'e-mails**.

## TL;DR

- **Email marketing ≠ cold outreach** : ici c'est du **marketing de permission (opt-in)**, data-driven, à fort ROI (Klaviyo annonce 63x sur l'email).
- **Klaviyo** = données-first / **B2C / e-commerce** (CDP native, prédictif CLV/churn, Shopify, service client, plan gratuit).
- **ActiveCampaign** = automation-first / **polyvalent B2B+B2C** (marketing automation le + flexible **+ CRM de vente intégré**, 900-1000+ intégrations).
- **Resend** = **API e-mail pour développeurs** (transactionnel-first, React Email, délivrabilité++), avec **Broadcasts** pour les newsletters ; facturé **au volume** (gratuit jusqu'à 3 000 e-mails/mois, puis 20 $/50k). **Pas** une suite marketing complète (ni CDP, ni SMS/WhatsApp, ni flows e-commerce).
- **Le juge de paix** : marketeur e-commerce data-driven → **Klaviyo** ; marketing + ventes B2B/multisecteur → **ActiveCampaign** ; développeur qui envoie par code (transactionnel + newsletters) → **Resend**.
