# Smartlead vs Instantly — Comparaison exhaustive & différences

> Face-à-face point par point. À lire après les fiches `01` (Smartlead) et `02` (Instantly). Relevé juin 2026.

---

## 1. En une phrase

- **Smartlead** = la plateforme **infrastructure-first / API-first**, pensée pour les **agences** et les power users qui veulent **contrôler la délivrabilité** et **construire par-dessus** (boîtes illimitées, serveurs dédiés, API réputée).
- **Instantly** = la plateforme **AI-first / simplicité-first**, pensée pour **trouver des clients vite** avec des **agents IA** et une **UX accessible**, intégrant data + envoi + CRM + closing.

Les deux sont des concurrents directs, de prix proche sur l'entrée de gamme, avec un socle commun (**boîtes & warmup illimités**) mais deux **philosophies opposées** : *infrastructure & contrôle* (Smartlead) vs *IA & facilité* (Instantly).

## 3. Différences de philosophie (le vrai clivage)

| Dimension | Smartlead | Instantly |
|---|---|---|
| **ADN** | Infrastructure & délivrabilité, **API-first** | IA & facilité d'usage, **product-led** |
| **Public privilégié** | **Agences de lead gen**, power users techniques | Founders, freelances, sales, débutants → mais aussi agences |
| **Promesse centrale** | « Possède ton infra, contrôle ta réputation » | « Find Clients » — l'IA fait le travail |
| **Courbe d'apprentissage** | Plus technique, dense | Plus douce, « intuitive » (point fort des avis) |
| **Approche IA** | Agents no-code **orientés workflow/ops** (alertes, sync, reporting) | Agents **orientés vente autonome** (SDR + Reply Agent qui closent) |
| **Extensibilité** | Très forte (écosystème bâti sur l'API : OutboundSync, etc.) | Forte côté **no-code/Automations** et intégrations |
| **Infra d'envoi** | Serveurs **dédiés single-tenant** (contrôle total) | **Sharding/rotation auto (SISR)** + **inbox maison (AirMail)** |

## 4. Comparaison tarifaire

> Attention : **les modèles de pricing ne sont pas directement comparables**. Smartlead = 1 plan global (contacts + sends) + add-ons. Instantly = **3 axes séparés** (Sending / Credits data / VIP) qui se cumulent.

### Entrée de gamme

| | Smartlead | Instantly |
|---|---|---|
| **1er plan payant** | **Base 39 $/mois** (2 000 contacts, 6 000 sends) | **Growth 47 $/mois** (1 000 contacts, 5 000 emails) — *axe Sending* |
| Boîtes illimitées | ✅ | ✅ |
| Warmup illimité | ✅ | ✅ |

### Milieu / scale

| | Smartlead | Instantly |
|---|---|---|
| Plan « populaire » | **Unlimited Smart 174 $** (contacts illimités, 150K sends) | **Hypergrowth 97 $** (25K contacts, 100K emails) |
| Haut de gamme self-serve | **Unlimited Prime 379 $** (500K sends, 3 serveurs, 3 clients, manager dédié) | **Light Speed 358 $** (500K emails, SISR) |
| Annuel | **-17 %** | **≈ -20 %** |

### Coûts « cachés » / add-ons à intégrer

| Brique | Smartlead | Instantly |
|---|---|---|
| Données / leads | SmartProspect « gratuit » (1 lead/3 emails) OU crédits vérif | **Instantly Credits** = axe payant séparé (Growth 47 $ → Hyper 197 $+) |
| Test délivrabilité | **SmartDelivery payant** (49-599 $) | **Inbox Placement inclus** |
| Infra dédiée | SmartServers **39 $/serveur** | SISR inclus dès Light Speed |
| White-label | **29 $/client** | inclus dans VIP (pas d'équivalent self-serve direct) |
| Vérification | add-on crédits | incluse |
| DFY complet | services / SmartSenders à l'usage | **VIP 2 000-10 000 $/mois (split 50/50)** |

**Lecture rapide :** sur le **pur envoi**, Smartlead Base (39 $) est un poil moins cher qu'Instantly Growth (47 $) et offre plus de sends (6 000 vs 5 000) et de contacts (2 000 vs 1 000). Mais **chez Instantly, la donnée (leads) est un axe payant à part** : un usage complet « leads + envoi » peut vite cumuler deux abonnements. Chez Smartlead, **plus de capacités sont des add-ons** (SmartDelivery, serveurs, white-label, vérif).

## 5. Différences sur la délivrabilité & l'infrastructure

| Aspect | Smartlead | Instantly |
|---|---|---|
| **Modèle d'infra** | Serveurs/IP **dédiés single-tenant** (SmartServers), clusters US+EU, 99,98 % uptime, failover | **SISR** : sharding + **rotation auto des IP** (swap des IP flaguées), Private Deliverability Network (Enterprise) |
| **Inbox propriétaire** | Non — revend Gmail/Outlook/SMTP (Zapmail, InfraInbox, Mailreef, AeroSend) | **Oui — AirMail** (infra maison, 4 $/inbox, indépendante de Google/Outlook, 50B+ sends de data) |
| **Warmup** | Pool **privé invite-only « reward-based »**, IA d'interactions, auto-pause/self-healing | **Navigateurs headless** + **pool 1M+ comptes réels**, read emulation, slow ramp |
| **Test de placement** | SmartDelivery (Placement Rate, 400+ blacklists, SpamAssassin) — **add-on** | Inbox Placement (400+ blacklists, SpamAssassin) — **inclus** |
| **Monitoring blacklists** | Temps réel + auto-alertes (SmartInfra) | Inclus (Inbox Placement) + triggers correctifs |

**Différence clé :** Smartlead mise sur **l'isolation dédiée** (vous « possédez » serveurs et IP). Instantly mise sur **l'automatisation invisible** (le SISR gère et fait tourner les IP pour vous) **et** sur une **infra d'inbox propriétaire (AirMail)** qui s'affranchit de la dépendance à Google/Outlook — un angle qu'aucun produit Smartlead n'a (Smartlead reste revendeur de boîtes Google/Outlook/SMTP).

## 6. Différences sur l'IA

| Aspect | Smartlead (SmartAgents) | Instantly |
|---|---|---|
| **Nature** | Agents **no-code orientés ops/workflow** | Suite d'**agents de vente autonomes** |
| **Exemples** | Alertes reply rate, push leads LinkedIn, monitor de réponses stale, sync Airtable, reporting clients | **AI Sales Agent** (trouve, écrit, relance, book — setup 30s via URL), **AI Reply Agent** (répond <5 min, Human-in-loop/Autopilot), **Automations** (no-code en langage naturel) |
| **Assistant chat** | Prompt library / SmartAI bot | **Copilot** « ChatGPT for outreach » + **WARP mode** (campagne complète) + Copilot Tasks/Memory |
| **Orientation** | Faire tourner les **opérations** d'une agence à l'échelle | **Remplacer le SDR** et le **suivi des réponses** |

**Différence clé :** l'IA d'Instantly va **plus loin dans l'autonomie commerciale** (un agent qui prospecte ET répond ET book seul, avec un mode Autopilot). Les SmartAgents de Smartlead sont surtout des **automatisations d'opérations** (monitoring, sync, reporting) — très utiles pour une agence, mais moins « SDR autonome » dans le discours.

## 7. Différences sur la data & le sourcing

| Aspect | Smartlead | Instantly |
|---|---|---|
| Base | SmartProspect **300M+** | Lead Database **450M+** (chiffres variables 160M-450M) |
| Modèle | **Leads « gratuits »** (1 vérifié / 3 emails envoyés) | **Crédits payants** (axe de pricing séparé) |
| Vérification | Waterfall **3X** (3 sources) | Waterfall **5+ providers**, catch-all (+40 % leads) |
| Enrichissement | Via **Clay** (partenariat) + SmartProspect | Natif : firmographics, job postings, news, tech, funding + 5 LLM + clé perso |
| Visiteurs site | ❌ | ✅ **Website Visitors / Pixel** |

**Différence clé :** Instantly intègre une **vraie brique data/intelligence autonome** (enrichissement multi-signaux, déanonymisation de visiteurs, agents de recherche IA), facturée en crédits. Smartlead s'appuie davantage sur **Clay** et un modèle de leads « offerts » lié au volume d'envoi.

## 8. Ce que l'un a et pas l'autre (différenciateurs nets)

**Uniquement (ou nettement plus poussé) chez Smartlead :**
- **White-label self-serve complet** sous votre domaine (`app.tondomaine.com`, 29 $/client) — argument agence majeur.
- **SmartDialer** dédié (parallel dialing, présence locale, AI coaching d'appels).
- **Serveurs dédiés single-tenant** que l'on « possède » (SmartInfra/SmartServers).
- **API/webhooks** réputés best-in-class (écosystème de produits tiers bâtis dessus, 25 clés nommées).

**Uniquement (ou nettement plus poussé) chez Instantly :**
- **AirMail** : infrastructure d'inbox **propriétaire** (non Google/Outlook).
- **AI Sales Agent** + **AI Reply Agent** (vente autonome, Human-in-loop/Autopilot).
- **Copilot / WARP mode** (assistant conversationnel central).
- **Website Visitors (Pixel)** : déanonymisation des visiteurs du site.
- **CRM avec Calls + SMS** intégrés dans le navigateur.

## 9. Quel outil pour quel profil ?

| Profil | Recommandation | Pourquoi |
|---|---|---|
| **Agence lead gen** gérant beaucoup de clients | **Smartlead** | White-label, workspaces clients, serveurs dédiés par client, API, coûts marginaux bas |
| **Founder / solo / débutant** voulant des clients vite | **Instantly** | UX simple, AI Sales Agent (setup 30s), Copilot, tout-en-un |
| **Power user technique** qui veut tout automatiser via API | **Smartlead** | API/webhooks best-in-class, 25 clés, écosystème |
| **Équipe sales** voulant automatiser la prospection ET le suivi des réponses | **Instantly** | AI Sales Agent + AI Reply Agent + Automations |
| **Obsédé délivrabilité** voulant contrôler son infra | **Smartlead** (serveurs dédiés) ou **Instantly AirMail** (inbox maison) | Deux approches différentes du même problème |
| **Besoin data/sourcing fort** (leads + intent + visiteurs site) | **Instantly** | Lead Database, enrichissement multi-signaux, Pixel |
| **Multicanal email + appels structuré** | **Smartlead** (SmartDialer) | Dialer dédié avec coaching IA |

## 10. Synthèse finale

Les deux plateformes couvrent désormais **toute la chaîne outbound** (data → envoi → délivrabilité → inbox → CRM → IA) et partagent le socle **boîtes + warmup illimités**. Le choix se joue sur **la philosophie**, pas sur une liste de features manquantes :

- Prenez **Smartlead** si votre priorité est le **contrôle de l'infrastructure**, la **délivrabilité à grande échelle**, le **white-label agence** et l'**automatisation par API**.
- Prenez **Instantly** si votre priorité est la **simplicité**, l'**IA de vente autonome**, la **donnée intégrée** (leads + visiteurs) et une mise en route **immédiate**.

Beaucoup d'agences avancées utilisent même **les deux** selon les clients/usages. Le test décisif reste toujours le même : **la délivrabilité sur VOS domaines** — d'où l'importance des outils de placement (SmartDelivery / Inbox Placement) avant de scaler.
