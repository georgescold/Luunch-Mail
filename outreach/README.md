## TL;DR

- **Socle commun** : boîtes mail illimitées + warmup illimité chez les deux.
- **Smartlead** = infrastructure-first / API-first / agences (serveurs dédiés, white-label, API best-in-class).
- **Instantly** = IA-first / simplicité-first (AI Sales Agent, Copilot, AirMail, data intégrée).
- **Entrée de gamme** : Smartlead Base **39 $/mois** vs Instantly Growth **47 $/mois**.
- **Le vrai juge de paix reste la délivrabilité sur vos propres domaines.**
